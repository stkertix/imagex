## Instalasi

```bash
npm install
```

## Menjalankan Aplikasi

```bash
npm run dev
```

Aplikasi akan berjalan di `http://localhost:3000`

## Build untuk Production

```bash
npm run build
```

## Preview Build

```bash
npm run preview
```

## Teknologi yang Digunakan
